# Welcome to Division

Division Engine is a project developed by Rex Woodfield. It is comprised of multiple overlapping parts.
- The game engine is public right now, and you can clone it and try it as much as you like, as long as you understand the license agreements and alpha status of the project.
- Division Translate is an upcoming C# to HLSL translation package meant to outperform ComputeSharp
- Division Math is an upcomping vectorized math library inspired by ComputeSharp and Unity.Mathematics, but expanded to have 1 to 1 functional parity with HLSL on the CPU and GPU.
